# ca-accounting-env
Financial Accounting RL Environment for  Meta PyTorch OpenEnv Hackathon. Simulates  real-world CA tasks — P&amp;L (Easy), Budget  Variance (Medium), Depreciation SLM/WDV  (Hard). Full OpenEnv spec with step(),  reset(), state() API. Graded rewards  0.0–1.0. Deployed on Hugging Face Spaces.
